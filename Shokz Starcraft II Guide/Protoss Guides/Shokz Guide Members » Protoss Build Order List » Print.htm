Protoss Build Order List

Posted By Shokz On July 29, 2010 @ 6:33 pm In Protoss Guides | Comments Disabled

A Build Order is the order in which you construct your buildings, units, or research upgrades. Build orders are often numbered by the supply count at which something is built, but can also simply be based in chronological order. For example, a “10 gate” is a gateway made when you have 10 probes. This could be followed up with a cybernetics core “immediately afterward”, meaning as soon as the gateway is completed, but at no particular supply count.

This is just the start and I will be adding a lot more build orders each day.

Standard Protoss 2 Gateway Build Order

9/10 -> Build a Pylon

13/18 -> Build your first Gateway

14/18 -> Build a Assimilator, once it is finished put 3 probes harvesting gas

15/18 -> Build a Pylon

18/26 -> Build a Cybernetics Core

18/26 -> Build a Assimilator, once it is finished put 3 probes harvesting gas

20/26 -> Build your second Gateway

21/26 -> Research Warpgate Technology from the Cybernetics core

Standard Zealots into Stalker Build Order

8/10 -> Build a Paylon

10/18 ->Send out probe to scout your opponent

12/18 -> Bulid a Gateway

14/18 -> Build a  Assimilator, once finished add 3 probes to gas

14/18 -> Build a Pylon

15/26 -> Build a Cybernetics core

17/26 -> Build your Second Gateway

18/26 -> Build a Pylon

You don’t want to straight tech to Stalkers, you’ll want 3-4 zealots out first before you start pumping them just in case of early rush.

After your cybernetics core is finished and start producing stalkers, once you you have a couple Stalkers / Zealots move in for the quick attack.

Void Ray Rush Build Order

10/10 -> Build a Pylon

10/18 -> Build a Gateway

10/18 -> Build an Assimilator, as soon as the Assimilator is finished put 3 Probes harvesting Gas

10/18 ->   Send a Probe to scout unit’s location if on 2v2 Map.

13/18 ->  As soon as the Gateway is finished make Cybernetics Core

14/18 -> As soon as Cybernetics Core is finished build Starport

Once your Starport is finished, immediately start building your first Void Ray; set the rally point  to your opponent’s base. Make sure that you use Chrono Boost on your Starport while your Void Ray is in production. If you are looking to take out their main base, then you will want to wait till you get 2 — 3 Void Rays so that you can fly right into your opponents base, and take it out.

Gateway + Forge Fast Expand Build Order

The gate+forge fast expand is a very strong and common build that isn’t covered by the 2 gate template. The general idea in a gate+forge fast expand is to build your first pylon, gate way, and forge by the choke into your natural, to protect yourself from early rushes, build a cannon or two, and then expand.

9/10 -> Build a Pylon

13/18 -> Build a Gateway

15/18 -> Build a Forge

15 /18 -> Build a Pylon

Produce any some zealots / cannons if needed

18+ Build a Nexus at your natural expansion

You will also want to use the early zealots to put on any kind of pressure you can, but being careful to not be overly aggressive and lose zealots unnecessarily.
