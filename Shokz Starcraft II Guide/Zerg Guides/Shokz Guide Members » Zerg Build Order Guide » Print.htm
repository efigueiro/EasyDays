Zerg Build Order Guide

Posted By Shokz On May 18, 2010 @ 6:07 pm In Zerg Guides | No Comments

This is a list of different build orders. I will be updating this list and adding the latest and greatest build orders.

Standard Zerg Start Build Order

This build order is a standard Zerg starting build.

In this starting build I normally take a drone around 7 or 8 supply and go scout, if you would like you can just use an overlord.

6/10 -> Build a Drone

7/10 -> Build a Drone

8 /10 -> Build a Drone

9/10 -> Overlord

10/18 -> Build a Drone

11/18 -> Build a Drone

13/18 -> extractor, once finished put 2 drones on gas.

13/18 -> Build a Spawning pool

13/18 -> Build a Queen

Once you have your queen out you can continue with a normal build order into Roaches, Hydras, or any other build you plan on going for the game.

Zerg vs Zerg Roach/Baneling Build Order

This Build is to counter Speedlings in a Zerg vs Zerg matchup.

9/10 -> Build an Overlord

13/18 -> Build a Spawning Pool

13/18 -> Build a Extractor and add 3 drones to it once finished.

16/18 -> Build a Queen

17/18 -> Use Extractor Trick

18/18 -> Build a Roach Warren

17/18 -> Build a Banelings Nest

17/18 -> Build an Overlord

They key to this build is to quickly produce Roaches with a couple of Banelings to counter a Speedling attack. If you can hit a group of Speedlings with your Banelings than you can take them out really easily and Roaches are a great counter to them also.
